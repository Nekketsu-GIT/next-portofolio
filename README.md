This is a [Next.js](https://nextjs.org/) portofolio. It uses sanity as CMS.

## First install packages

```bash
npm i
```

## Config

First create a .env.local file and paste into the content of .env.local.dist. Then add your own credential

## Storybook
```bash
npm run storybook
```

## RUN Locally

```bash
npm run dev
```
